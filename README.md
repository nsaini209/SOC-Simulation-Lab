# SOC-Simulation-Lab

## Objective

This project was to design and deploy a SOC environment capable of automating the full incident response lifecycle. The primary goal was to integrate Wazuh (SIEM) and TheHive (Case Management) via Shuffle (SOAR) to transform raw endpoint telemetry into enriched security alerts.

### Skills Learned
- SOAR Automation: Designing automated workflows in Shuffle to bridge detections with response actions.
- Endpoint Monitoring: Deploying and configuring Sysmon and Wazuh Agents on Windows 11 to capture granular telemetry.
- Database Management: Configuring Cassandra for data persistence and Elasticsearch for high-speed log indexing and search.
- Config Management: Modifying complex .yml, .conf, and .yaml files to bind services to correct IP addresses and listeners.
  
### Tools Used
- Wazuh
- Shuffle
- TheHive
- DigitalOcean
- ElasticSearch
- Cassandra
- VirusTotal
